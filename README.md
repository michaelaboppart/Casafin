# CasaFin — Swiss Financial OS for Families

> Endlich Überblick über dein Geld.

## Live
- Production: [casafin.ch](https://casafin.ch)
- Preview: Vercel Auto-Deploy from `main`

## Stack
- Vanilla HTML/CSS + React JSX components
- Bilingual DE/EN (data-en/data-de pattern)
- Stripe Payment (Pioneer CHF 199/year)
- Swiss-Native: 26 Cantons, 3. Säule, AHV/BVG/Quellensteuer

## Files
- `CasaFin Landing.html` — Premium marketing site
- `CasaFin.html` — Main SPA application
- `CasaFin Admin.html` — Admin dashboard
- `CasaFin Secure App.html` — Auth/security layer
- `screens-*.jsx` — 13 React screen modules (Dashboard, Money, Life, Fina, Budget, Auth, Settings, Docs, Business, Agents, Accounts)
- `i18n.js` — Bilingual i18n engine
- `app.css` — Design system (Hanken Grotesk + JetBrains Mono, Teal #0B8F92 + Gold #C2862A)

## Brand
- Tagline: "Endlich Überblick über dein Geld" / "Finally, clarity over your money"
- AI Assistant: **Fina**
- Pioneer Offer: CHF 199/year for life (first 100 customers)
- Targets: Swiss families with CHF 200k+ flow
